Instructions

Use Java or C++ to implement a solution to the problems listed below. When you are done, come pass it off the one of the judges. You have from now until 8:30 P.M. Most points win. Good luck.

Problems

Problem 1 (3 points)

Given an input read from stdin, print it backwards and diagonally.

Specifications

Input: A string
Output: The string printed diagonally and in reverse

Output:

 t

  e

   P

    

     l

      o

       t

        s

         i

          P

Problem 2: Caesar Cipher (3 points)

The caesar cipher is a (very outdated...) method of encryption that uses a key, also known as a shift, to encrypt messages.
To encrypt any letter of the alphabet, you simply add the shift to it. For example, with a shift of 3, a becomes a + 3 which is d, b becomes e, etc. Some more info on the notes from our cryptography workshop
Brutus wants to figure out what key Caesar most commonly uses. He somehow has access to several messages and their encrypted counterparts. Write a program to help Brutus figure out what key was used to encrypt a certain message.

Specifications

Input1: a plaintext String
Input2: another string that is the encrypted form of Input1
Output: The key used to encrypt Input1 into Input2
Assume that only letters a-z will be in the input. No need to worry about punctuation or capital letters.
The encryption scheme wraps around, i.e. z + 3 = c, c + 25 = b, z + 20 = t
You can assume that Input2 will always be the correct encrypted form of Input1.
Note: The ascii range for a-z starts with a at 97. If you want to work with a-z as 1-26 subtract 96 from the char.

Example Run

Plain text: gopokes
Encrypted text: ltutpjx
Key Used: 5

Problem 3: The rorrim mirror (3 points)

Given an integer n, and a list of n words, print out n lines. Each line should have one of the words from the list, first printed backwards, then printed forwards, with one space inbetween. On any given line, there must be enough leading spaces so that the first character of the words line up on the printout.

Specifications

Input: Integer n, followed by n words
Output: Exactly as specified in problem

Example Run

Input: 4 Sun Moon Stellar Balloon
Output:

    nuS Sun

   nooM Moon

ralletS Stellar

noollaB Balloon

Problem 4: So many integer sequences... (3 points)

The fibonacci sequence F(n) is defined as follows:
F(n) = F(n-1) + F(n-2) with F(0) = 0 and F(1) = 1 (more info)
Recamán's sequence is defined as follows:
a(0) = 0; for n > 0, a(n) = a(n-1) - n if a(n-1)-n is positive and not already in the sequence, otherwise a(n) = a(n-1) + n. (more info)
Write a program that, when given a number n, determines if n is in the the first 45 elements of the fibonacci sequence. If n is not in the first 45 elements of the fibonacci sequence, tell the user so and stop the program. Otherwise, figure out which index of the fibonacci sequence n is, and determine the value of Recamán's sequence for that index.

Specifications

n <= 701408733

You only have to check if n is one of the first 45 elements of the fibonacci sequence, that is fib(0) through fib(44).

Example Runs

Input: 4181
Output: 62
Input: 701408733
Output: 36
Input: 9
Output: 9 is not in the first 45 elements of the fibonacci sequence
