Programming Competition

October 4, 2017

Instructions

Use Java, C, C++, or Python (2 or 3) to implement a solution to the problems listed below. When you are done, raise your hand to pass it off to one of the judges or email your solution to okstateacm@gmail.com. You have from 5:30pm to 8:30pm. The most points win. Good luck.

Rules

  • All of OSU's academic integrity rules apply. You are free to use Google/your friends so long as your code is your own.
  • Unless the problem specifies otherwise, all input will be read from stdin (so, for Java this means a Scanner reading System.in)
  • You must pass your code off (or at least have had your hand up) before 8:31 P.M. in order to get credit for the problem.
  • If you come to one of the judges to pass off a program and it doesn't work, you may fix it and come back as many times as necessary, given it is still before 8:31 P.M.
  • If any problem seems too ambigous, ask a judge to clarify.
